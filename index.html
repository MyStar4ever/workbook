Дневник эмоций и мыслей

Краткая инструкция: отмечай эмоции и ставь процент интенсивности — это помогает увидеть, что именно доминирует. В блоке «Мысли» записывай автоматические негативные мысли, разбирай их по шагам (искривления → объяснение → новые рациональные мысли). Всё сохраняется автоматически в браузере.

Эмоции

Клик по названию — выбирает эмоцию. Клик по проценту — редактировать.

Негативные мысли — разбор

Порядок: мысль → искажения → объяснение → новые рациональные мысли. Можно добавлять несколько рациональных ответов.
